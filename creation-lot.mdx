## Objectif

Cette fonctionnalité permet de configurer un bien complet en cinq étapes guidées : identité, spécificités, adresse, critères de candidature et finalisation. Elle constitue la base de tout le parcours locatif puisque c'est elle qui filtre automatiquement les candidatures recevables.

<Note>
  **Où trouver cette fonctionnalité ?** Dashboard \> **Configurer un nouveau lot** ou **Création de lot** — ou depuis une fiche lot \> **Modifier le lot**.
</Note>

## Erreurs fréquentes à éviter

<Warning>
  - Saisir le loyer hors charges : Uplace calcule les seuils de revenus sur le loyer total CC.
  - Mettre « Garant : Refusé » par erreur, ce qui élimine de bons dossiers avec garant.
  - Mettre « Garant : Obligatoire », ce qui élimine de bons dossiers sans garant.
  - Oublier d'attribuer un responsable, ce qui empêche un suivi clair des candidatures.
  - Activer la validation automatique sur un lot atypique aux critères mal calibrés.
</Warning>

## Ce que fait Uplace automatiquement

<Info>
  - Calcule le salaire minimum requis selon le multiplicateur choisi (ex. x3 pour 798 € → 2 394 €).
  - Applique les critères à toutes les candidatures pour les classer (Excellent / Bon / Mauvais).
  - Pré-remplit la GLI sélectionnée sur les futures candidatures du lot.
  - Ferme automatiquement les nouvelles candidatures si la capacité maximum est atteinte.
</Info>
